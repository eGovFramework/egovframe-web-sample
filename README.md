## 프로젝트 실행

1. eclipse 하단의 Servers 탭을 클릭하고, 마우스 우클릭하여 **New > Server** 를 선택하여 서버를 설치한다.

2. 실행할 프로젝트를 마우스 우클릭하고 **Run As > Run on Server** 를 선택한다.
